# サーバー一覧

このページでは、利用可能なサーバーの一覧をまとめています。該当する行の後ろにある **Copy** ボタンをクリックすると、クリップボードにコピーすることができます。

サーバーが利用できない場合は、別のサーバーをお試しください。

## どう使ったらいいの？

**[Sonolusのクイックスタート](guide/quick-start)**をよく見てください。 サーバーによっては、ブラウザで直接開くと説明書を見れる場合もあります。

*補足: **日本東京の加速路線**は特に利用者が多いため、月末になると通信転送上限に達してしまい、利用できなくなる可能性があります。 その場合は、別のサーバーをご利用ください。 *
